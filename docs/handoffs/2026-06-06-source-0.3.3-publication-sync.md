## Why swarm was briefly behind source

Release preparation and publication receipts are source-side work by policy
(`docs/contributing/SWARM_TO_MAIN.md` routing). During publication, source
gained the catch-up merge commit (#526), the release prep commit (#527), and
the receipt commit (#528) that swarm did not have. This mirror restores parity
and the checkpoint records that the divergence is acknowledged and absorbed.

## Trust boundary
