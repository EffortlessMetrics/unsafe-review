## Trust boundary

`0.2.1` remains an experimental advisory release:

- no memory-safety proof,
- no UB-free claim,
- no Miri-clean claim,
- no site-execution claim,
- no calibrated precision or recall claim,
- no witness execution by default,
- no automatic comments,
- no source edits,
- no default blocking policy.

## Known limits

- This release changes package-visible presentation and badge semantics only.
- It does not add analyzer behavior.
- It does not promote coverage badges into source.
- It does not make live LSP, VS Code, witness execution, comment posting, or
  blocking policy part of the public default flow.
