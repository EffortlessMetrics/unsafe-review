## Release framing

`0.2.0` is the public-usability release:

- install the facade crate,
- run `doctor`,
- run one `first-pr` / `review` command,
- open the PR summary,
- explain one `ReviewCard`,
- inspect witness routes and saved projections without overtrusting them.

`0.2.0` remains an **experimental advisory** release:

- no memory-safety proof,
- no UB-free claim,
- no Miri-clean claim,
- no site-execution claim,
- no witness execution by default,
- no automatic comments,
- no source edits,
- no default blocking policy.

Live LSP runtime and VS Code publication remain deferred. The 0.2.0 editor
surface is saved `lsp.json`.

## Stop conditions

Do not publish if:

- crates.io reports any of the three `0.2.0` versions already published,
- the working tree is dirty,
- `cargo publish -p unsafe-review-core --dry-run` fails,
- any downstream dry-run fails for a reason other than the expected unpublished
  dependency version,
- README, badge endpoint, support-tier, first-run, LSP, agent-packet, receipt,
  policy, or release wording implies safety, UB-free status, Miri-clean status,
  site execution, automatic comments, source edits, or policy readiness.
