## Publish method note (new for 0.3.7)

0.3.7 is the first release published via the `crates-publish` GitHub Actions
workflow on `EffortlessMetrics/unsafe-review`. The workflow publishes in
dependency order with sparse-index retry loops. The org `CARGO_REGISTRY_TOKEN`
secret grants the source repo publish access. The swarm already carries the
workflow (PR #1721, `de5382e4`) — the workflow is NOT brought in again via
this mirror (it is already present).

## Why swarm was behind source

Release preparation and publication are source-side work by policy. During
publication, source gained the catch-up import + release-prep merge (#536),
the crates-publish workflow port (#537), and the receipt (#538) that swarm
did not have. This mirror restores parity and the checkpoint records that the
divergence is acknowledged and absorbed. See
`docs/contributing/SWARM_MIRROR.md` for the mirror-back runbook.
