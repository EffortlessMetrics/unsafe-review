## What this release carries

Branch `sync/swarm-0.3.4-catchup` imports swarm main `3805a0de`
(history-preserved, tree byte-identical to validated swarm main), covering the
post-0.3.3 coverage-instrument lane recorded under `CHANGELOG.md` `0.3.4`:

- SPEC-0029 unsafe-evidence coverage block: slot-based model assigning each
  `unsafe` site a coverage slot, tracking which slots have evidence,
- SPEC-0030 baseline coverage-movement keystone: baseline recording, movement
  tracking, diff-scoped `no-new-debt`, `worsened_gaps`, `baseline init` and
  `baseline add` authoring subcommands,
- SPEC-0031 baseline-aware badge: coverage badge reflects baseline movement
  state without implying proof,
- SPEC-0032 comment-plan coverage-gap hardening: comment plans anchored to
  coverage gaps, phantom-anchor elimination,
- SPEC-0033 file:range context scan: `context` subcommand emits a `file:range`
  context packet for LLM and agent consumers,
- SPEC-0034 `unsafe-review-gate.json` routing manifest: structured gate
  manifest written by every `first-pr` run,
- SPEC-0035 repo-scan diagnosability: scan-scope metadata distinguishes
  scanned-but-empty from not-scanned,
- stale JS-buffer span detection: analyzer flags span use after GC-reentry,
- stable-byte coverage v1 with span-as-arg detection and snapshot suppression,
- candidate `new` and `lint` authoring subcommands,
- opt-in `confirm --allow-heavy` cue executor with `runtime_executed`
  projection,
- confirmation state projection and cheapest-confirmation ranking,
- dogfood usefulness rollup with drift rail,
- first-run `doctor` readiness check,
- CI single tight gate with full GitHub-hosted fallback by default.

## Trust boundary

`unsafe-review` remains advisory static coverage evidence. It provides
baseline-aware unsafe evidence coverage artifacts for repo badges, PR gates,
line-comment plans, LSP/LLM context, and ub-review. It does not prove memory
safety, UB-free status, Miri-clean status, site execution, calibrated
precision/recall, or policy readiness, and it does not run witnesses, post
comments, edit source, or block by default.
