# Accuracy error taxonomy

| Code | Meaning |
|---|---|
| `INV-MISS` | missed unsafe site |
| `INV-SPURIOUS` | spurious site |
| `FAM-WRONG` | wrong operation family |
| `HAZ-MISS` | missing hazard |
| `HAZ-SPURIOUS` | extra hazard |
| `EVID-FP` | invalid evidence accepted |
| `EVID-FN` | valid evidence rejected |
| `ROUTE-WRONG` | wrong witness route |
| `ID-DRIFT` | identity drift |
| `OVERCLAIM` | artifact overclaim |
